# 快捷键

> **Mod 键**: 本地 TTY → `Super`；VM/嵌套窗口 → `Alt`

## Kitty

### 窗口/标签页

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+T` | 新建标签页 |
| `Ctrl+Shift+W` | 关闭标签页 |
| `Ctrl+Shift+Enter` | 上下分割窗口 |
| `Ctrl+Shift+\` | 左右分割窗口 |
| `Ctrl+Shift+方向键` | 切换窗口焦点 |
| `Ctrl+Shift+L` | 切换布局 |

### 标签页导航

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+1~9` | 切换到第 N 个标签页 |
| `Ctrl+Shift+Left/Right` | 上/下一个标签页 |

### 字体

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+=` | 增大字体 |
| `Ctrl+Shift+-` | 减小字体 |
| `Ctrl+Shift+0` | 重置字体大小 |

### 滚动

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+Page_Up/Down` | 向上/下滚动 |
| `Ctrl+Shift+Home/End` | 滚动到顶部/底部 |

### 复制/粘贴

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` | 复制到剪贴板 |
| `Ctrl+Shift+V` | 从剪贴板粘贴 |

## Niri

### 常用

| 快捷键 | 功能 |
|--------|------|
| `Mod+Shift+/` | 快捷键覆盖层 |
| `Mod+T` | 终端 (kitty) |
| `Mod+D` | 应用启动器 (fuzzel) |
| `Mod+Shift+L` | 锁屏 (hyprlock) |
| `Mod+O` | 概览 |
| `Mod+Q` | 关闭窗口 |

### 焦点

| 快捷键 | 功能 |
|--------|------|
| `Mod+←↓↑→` | 焦点移动 |
| `Mod+Home/End` | 焦点移至列首/列尾 |

### 移动窗口

| 快捷键 | 功能 |
|--------|------|
| `Mod+Ctrl+←↓↑→` | 移动窗口/列 |
| `Mod+Ctrl+Home/End` | 移至首/末位 |

### 显示器

| 快捷键 | 功能 |
|--------|------|
| `Mod+Shift+←↓↑→` | 切换显示器焦点 |
| `Mod+Shift+Ctrl+←↓↑→` | 将列移至目标显示器 |

### 工作区

| 快捷键 | 功能 |
|--------|------|
| `Mod+1~9` | 切换工作区 |
| `Mod+Page_Down/Page_Up` | 上/下工作区 |
| `Mod+Ctrl+1~9` | 将列移至工作区 |
| `Mod+Ctrl+Page_Down/Page_Up` | 将列移至上/下工作区 |
| `Mod+Shift+Page_Down/Page_Up` | 移动工作区顺序 |

### 布局

| 快捷键 | 功能 |
|--------|------|
| `Mod+R` | 切换列宽预设 |
| `Mod+Shift+R` | 切换窗口高度预设 |
| `Mod+Ctrl+R` | 重置窗口高度 |
| `Mod+F` | 最大化列 |
| `Mod+Shift+F` | 全屏窗口 |
| `Mod+Ctrl+F` | 扩展至可用宽度 |
| `Mod+C` | 居中当前列 |
| `Mod+Ctrl+C` | 居中所有可见列 |
| `Mod+-/=` | 列宽 -/+10% |
| `Mod+Shift+-/=` | 窗口高度 -/+10% |

### 窗口操作

| 快捷键 | 功能 |
|--------|------|
| `Mod+V` | 浮动/平铺切换 |
| `Mod+Shift+V` | 浮动与平铺间切换焦点 |
| `Mod+W` | 列标签页显示模式 |
| `Mod+[` | 窗口吸入左侧列 / 弹出 |
| `Mod+]` | 窗口吸入右侧列 / 弹出 |
| `Mod+,` | 右侧窗口吸入当前列下方 |
| `Mod+.` | 当前列底部窗口弹出到右侧 |

### 截图 / 媒体 / 其他

| 快捷键 | 功能 |
|--------|------|
| `Mod+Shift+G` | 区域截图 → 剪贴板 |
| 音量键 | 音量 +/-10%，静音，麦克风静音 |
| 播放键 | 播放/暂停，停止，上一首，下一首 |
| 亮度键 | 亮度 +/-10% |
| `Mod+Escape` | 切换快捷键抑制 |
| `Mod+Shift+E` | 退出 Niri |
| `Ctrl+Alt+Delete` | 退出 Niri |
| `Mod+Shift+P` | 关闭显示器 |

### 滚轮

| 快捷键 | 功能 |
|--------|------|
| `Mod+滚轮↓↑` | 切换工作区 |
| `Mod+Ctrl+滚轮↓↑` | 将列移至工作区 |
| `Mod+滚轮←→` | 焦点左右移动 |
| `Mod+Ctrl+滚轮←→` | 列左右移动 |
| `Mod+Shift+滚轮↓↑` | 焦点左右移动 |
| `Mod+Ctrl+Shift+滚轮↓↑` | 列左右移动 |

## Neovim

### 普通模式

| 快捷键 | 功能 |
|--------|------|
| `<Space>w` | 保存 |
| `<Space>q` | 退出 |
| `<Space>e` | 文件树 |
| `jk` | 退出插入模式 |
| `<Space>ff` | 查找文件 |
| `<Space>fg` | 搜索内容 |
| `<Space>fb` | 搜索缓冲区 |
| `<Space>fh` | 搜索帮助 |
| `Ctrl+方向键` | 窗口导航 |

### Visual 模式

| 快捷键 | 功能 |
|--------|------|
| `J` | 向下移动选中行 |
| `K` | 向上移动选中行 |
| `>` | 向右缩进 |
| `<` | 向左缩进 |

### 补全 (cmp)

| 快捷键 | 模式 | 功能 |
|--------|------|------|
| `Tab` | 插入 | 下一个候选项 |
| `Shift+Tab` | 插入 | 上一个候选项 |
| `Enter` | 插入 | 确认选择 |
| `Ctrl+Space` | 插入 | 手动触发补全 |
| `Ctrl+E` | 插入 | 关闭补全菜单 |
| `Ctrl+B` | 插入 | 文档向上滚动 |
| `Ctrl+F` | 插入 | 文档向下滚动 |
